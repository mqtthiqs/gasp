\subsubsection{Typing annotation and argument naming}

The first step in the translation is the embedding of \XLF\ into a
language with two small variations:
\begin{itemize}
\item application spines are annotated with their types, as well as
  metavariables and
\item each argument of such spines bears the name of the ``formal
  argument'' in the type of the head. For example, the function $t :
  \prd x A \prd y B C$ applied to terms $u$ and $v$ will be written
  $\laapp{t}{\lncons x u {\lnsing y v}} C$.
\end{itemize}

While the first point is easily justified by above arguments, the
second will be motivated in the next section. Informally, naming all
arguments with their expected variable in the type allow to forget
their order in a list $l$; a side-effect of this choice is discussed
below.

We thus modify the syntax of \XLF\ to obtain \XLFa (Fig. \ref{fig:XLFa}):

\begin{figure}
  \def\thelangmod{\XLFamod}
  \begin{align*}
    \mk &\gequal { \prd\mv\mf\mk
      \gor \type } \\
    \mf &\gequal { \prd\mv\mf\mmf
      \gor \memph{\laapp\mcf\ma\mk} } \\
    \mo &\gequal { \lam\mv\mf\mo
      \gor \memph{\laapp\mh\ma\mf}
      \gor \memph{\typ\mmeta\mf}
      \gor \obox\mo\mv\ms } \\
    \mh &\gequal { \mo
      \gor \mco
      \gor \mv } \\
    \ma &\gequal {\lnil
      \gor \memph{\lncons \mv\mo\ma}} \\
    \ma &\gequal { \lnil
      \gor \lcons\mo\ma } \\

\label{fig:XLFa}
\end{figure}

\begin{figure}
  \def\thelangmod{\XLFamod}
  \underline{Judgements}
  \begin{gather}
    \jannotK\repo\me {\underline K} {K'} \\
    \jannotA\repo\me {\underline A} {A'} \\
    \jannott\repo\me {\underline t} {t'} \\
    \jannoth\repo\me {\underline h} {h'} A \\
    \jannotl\repo\me {l'} A {\underline l} {l''} A' \\
    \jannots\repo\me {\underline\sigma} {\sigma'}
  \end{gather}
  \underline{Translation}
  \begin{mathpar}
    %% kind
    \infer{ }{\jannotK\repo\me\type\type}
    \and
    \infer{
      \jannotK\repo\me\mf\mmf \and
      \jannotK\repo{\ebinddecl\me\mv\mf} \mk\mmk
    }{
      \jannotK\repo\me{\prd\mv\mf\mk}{\prd\mv\mmf\mmk}
    }
    \and
    %% fam
    \infer{
      \jannotl\repo\me\lnil{\elook\Sigma a} l {l'} K
    }{
      \jannotK\repo\me{\lapp a l}{\laapp a {l'} K}
    }
    \and
    \infer{
      \jannotA\repo\me A {A'} \and \jannotA\repo{\ebinddecl\me x A}{B}{B'}
    }{
      \jannotA\repo\me{\prd x A B}{\prd x {A'}{B'}}
    }
    \and
    %% obj
    \infer{
      \jannotA\repo\me A {A'} \and \jannott\repo{\ebinddecl\me x {A'}} t {t'}
    }{
      \jannott\repo\me{\lam x A t}{\lam x {A'}{t'}}
    }
    \and
    \infer{
      \jannoth\repo\me h {h'} A \and
      \jannotl\repo\me\lnil A l {l'} {A'}
    }{
      \jannott\repo\me{\lapp h l}{\lapp{h'}{l'}}
    }
    \and
    \infer{
      \jannott{\nlfgo\repo p}\me t {t'} \and
      \jannott\repo\me\sigma{\sigma'}
    }{
      \jannott\repo\me{\obox t p\sigma}{\obox t p \sigma}
    }
    \and
    \infer{ }{
      \jannott\repo\me{\meta x}{\mameta {\nlflook\repo {\mmeta}}}
    }
    \and
    %% args
    \infer{  }{
      \jannotl\repo\me l A \lnil l A
    }
    \and
    \infer{
      \jannott\repo\me t {t'} \and
      \jannotl\repo\me{\lncons x {t'} {l'}} B l {l''} C
    }{
      \jannotl\repo\me{l'}{\prd x A B} {\lcons t l} {l''} C
    }
    \and
    %% head
    \infer{ }{
      \jannoth\repo\me x x {\elook\me x}
    }
    \and
    \infer{ }{
      \jannoth\repo\me c c {\elook\Sigma c}
    }
    \and
    \infer{ }{
      \jannoth\repo\me a a {\elook\Sigma a}
    }
    \and
    \infer{
      \jannott\repo\me t {t'}
    }{
      \jannoth\repo\me t {t'} {(\typeof {t'})}
    }
    \and
    %% subst
    \infer{ }{
      \jannots\repo\me\snil\snil
    }
    \and
    \infer{
      \jannott\repo\me t {t'} \and
      \jannots\repo\me \sigma{\sigma'}
    }{
      \jannots\repo\me{\scons \sigma x t}{\scons \sigma x {t'}}
    }
  \end{mathpar}

    \typeof{(\lam x A t)} &= \prd x A {(\typeof t)} \\
    \typeof{(\laapp h l A)} &= A
